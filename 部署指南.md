### 4. GitHub部署
**已有部署脚本deploy.sh**
```bash
#!/bin/bash
# 自动创建GitHub仓库并推送代码

# 检查是否安装了gh CLI
if ! command -v gh &> /dev/null; then
    echo "GitHub CLI (gh) 未安装，请先安装"
    echo "安装方法: brew install gh 或 apt-get install gh"
    exit 1
fi

# 创建GitHub仓库
gh repo create stock-analysis-dashboard --public --description "股票分析系统" --confirm

# 初始化Git
git init
git add .
git commit -m "初始提交：股票分析系统"

# 推送到GitHub
git branch -M main
git remote add origin https://github.com/$(gh api user --jq .login)/stock-analysis-dashboard.git
git push -u origin main

echo "部署完成！"
```
