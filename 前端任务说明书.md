# 前端任务说明书（给 @前端Bot）

## 项目背景
我们需要开发一个股票分析页面系统，需要前端提供用户界面、图表展示和交互功能。

## 具体任务

### 1. UI界面设计
设计以下页面布局：

- **主页面布局**：
  - 顶部导航栏（包含时间选择器）
  - 左侧板块选择面板
  - 中间主内容区域（图表展示）
  - 右侧个股展示面板
  - 底部AI建议区域

- **响应式设计**：适配手机、平板、电脑

### 2. 图表组件实现
实现以下图表组件：

- **板块分析图表**：
  - 每日板块分析柱状图
  - 每周板块分析折线图
  - 资金流向饼图/柱状图

- **联动展示图表**：
  - 板块联动关系图（网状图或关系图）
  - 个股滚动展示组件（卡片轮播）

- **预测展示图表**：
  - AI预测结果折线图
  - 选股建议列表

### 3. 交互功能实现
- **时间选择器**：切换每日/每周分析
- **板块筛选器**：选择特定板块查看
- **策略选择器**：选择不同投资策略
- **个股搜索**：搜索特定个股信息
- **数据刷新**：实时更新数据

### 4. API集成
- 调用后端API获取数据
- 处理API响应和错误
- 实现数据加载状态显示

### 5. 视觉效果
- 股票市场主题配色
- 数据可视化样式优化
- 动画过渡效果

## 技术要求
- 框架：React/Vue/纯HTML+JS
- 图表库：Chart.js/ECharts/D3.js
- 样式：CSS/Sass/Tailwind CSS

## 输出要求
- 完整的HTML页面
- 所有图表组件
- 响应式适配
- 良好的用户体验

## 进度要求
- 第1天：完成页面布局和UI设计
- 第2天：完成图表组件
- 第3天：完成交互功能和API集成

请回复确认接收任务，并提供你的开发计划。
